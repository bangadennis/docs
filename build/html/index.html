DHIS2 supply chain hierarchy project documentation¶

This is documentation for the hierarchy of the drugs supply chain that is not captured in DHIS2 for purposes of analysis and report generation.

There is a downloadable PDF version of this documentation, a mobile friendly EPUB version and a downloadable HTML version.
